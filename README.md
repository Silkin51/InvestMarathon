# Invest Marathon

3D-раннер про инвестирование: счёт — это жизнь, бег сжигает очки, а бонусы — инструменты с разным риском.

**Играть:** https://silkin51.github.io/InvestMarathon/

## Механика

- Старт с 500 очков; расход растёт со скоростью (линейный + квадратичный член + % от счёта).
- 🟢 **+100** — безопасный фикс; 🔵 **+3%** — безопасный множитель.
- 🟡 **+20% / −10%** — малый гэмбл; 🟠 **+100% / −50%** — крупная ставка.
- 🔴 Барьеры −100: объезжай, перепрыгивай (низкие) или подкатывайся (перекладины).
- Рядом бежит бот-конкурент с человеческой задержкой реакции (0.1–0.7 с) и собственным счётом.
- 6 полос; если сосед вплотную и идёшь в его полосу — герой перепрыгивает сразу через две.

## Управление

- `←` `→` (или `A`/`D`) — смена полосы
- `↑` / `W` / пробел — прыжок
- `↓` / `S` — подкат
- Свайпы на телефоне

## Технически

Один HTML-файл + Three.js r128 (лежит рядом, работает офлайн). Персонажи, город и монеты — процедурные, без внешних ассетов.
